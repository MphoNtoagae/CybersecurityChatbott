# 🔐 Cybersecurity Awareness Chatbot

## 📌 Description

This project is a C# console-based chatbot developed to promote cybersecurity awareness.
The chatbot interacts with users, provides basic cybersecurity tips, and demonstrates concepts such as user input handling, string manipulation, and structured programming.

## 🎯 Features

* 🎤 Voice greeting using a WAV file
* 🖥 ASCII art displayed on startup
* 👤 Personalized user greeting
* 💬 Interactive chatbot responses
* 🔐 Cybersecurity topics:

  * Password safety
  * Phishing awareness
  * Safe browsing
* ⚠ Input validation for incorrect or empty input
* 🎨 Enhanced console UI with colors and formatting


## ⚙️ Technologies Used

* C# (.NET Console Application)
* System.Media (for audio playback)
* GitHub (version control)

## ▶️ How to Run the Program

1. Open the project in Visual Studio
2. Build the solution
3. Run the program
4. Enter your name when prompted
5. Start interacting with the chatbot

## 📂 Project Structure

* Program.cs → Main entry point
* Chatbot.cs → Chatbot logic and interaction
* greeting.wav → Voice greeting file

## 🧪 Example Questions You Can Ask

* "How are you?"
* "What is your purpose?"
* "Tell me about passwords"
* "What is phishing?"
